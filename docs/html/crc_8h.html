uint8_t crc8 ( const uint8_t *  pcBlock,
size_t  len 
)

This function calculate checksum CRC-8-Dallas/Maxim.

  • Used polynomial: x^8 + x^5 + x^4 + 1 (0x31).
  • Init : 0xFF
  • Revert: false
  • XorOut: 0x00
  • Check : 0xF7 ("123456789")
Parameters
[in]pcBlockpointer to the data buffer.
[in]lensize of data buffer.
Returns
calculated crc8 value.
